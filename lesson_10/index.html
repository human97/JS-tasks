Block #10. Массивы в JavaScript (часть 1)

Изучаем массивы в javascript. Задачи на массивы

Task 1. Создайте массив содержащий строки, числа, булевы значения. Выведите его в консоль.

Task 2. Создайте массив содержащий строки, числа, булевы значения. Выведите его в div. Совет. Попробуйте синтаксиc elem.innerHTML = ваш_массив.

Task 3. Создайте массив a. Выведите длину массива a. 

const a = [2,'hello', 17, 34, 'privet']

Task 4. Создайте массив a. Выведите нулевой, третий, восьмой элемент массива a в консоль. Сделайте вывод о 8-м элементе. 

const a = [2, 'hello', 17, 34, 'privet']

Task 5. Создайте массив a. Выведите сумму нулевого, второго и третьего элементов массива. 

const a = [2, 'hello', 17, 34, 'privet']

Task 6. Создайте массив, который содержит ваше имя, рост, день рождения и месяц рождения. Выведите массив на страницу в div.

Task 7. Создайте массив b. Добавьте в него значения 'hi', 'foo', 'bar'. Вывести массив на страницу в элемент div. 

let b = ['one', 1, 2, 'two']

Task 8. Создайте массив b8. Добавьте в него третий элемент равный 3.14, 4 элемент равный 17, 6 элемент равный 5. Выведите массив и в консоль и на страницу. Обратите внимание, как выводятся элементы значения которых не заданы. Также выведите значение длины массива. 

let b8 = ['one', 1, 2, 'two']

Task 9. Создайте массив, который содержит 10 значений. Выведите 3-й и 7-й элемент. Выведите весь массив в консоль.

Task 10. Создайте массив содержащий только 1-й, 5-й и 10 элемент. Выведите его в консоль. Выведите его в div. Выведите длину данного массива.
