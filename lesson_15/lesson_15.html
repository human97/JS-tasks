Block #15. Set в JavaScript

Task 1. Создайте новый Set. Добавmте в него через add элементы. Выведите в консоль.

Task 2. Создайте input куда пользователь может вводить элементы и кнопку. По нажатию на кнопку значение из input добавляется в set. Set выводится в консоль.

Task 3. Добавьте к предыдущему заданию кнопку удалить. При нажатии кнопки читайте содержимое input и удаляйте его из Set. Если input пустой - выводите ошибку (alert).

Task 4. Добавьте к предыдущему заданию кнопку проверить. При нажатии кнопки читаете содержимое input и проверяет его наличие в Set. Функция должна возвращать true или false. Действия запускаются при вызове функции t4.

Task 5. Выведите размер полученного в результате предыдущих операций Set.

Есть массив arr6 = [3,4,3,2,4,56,1,23]. Напишите функцию, которая возвращает и выводит на страницу количество уникальных элементов в массиве arr6. Решение должно использовать set. Действия должны запускаться при вызове функции t6.

Task 7. Создайте input куда пользователь может ввести пароль и кнопку - проверить. Проверяйте с помощью Set, чтобы пользователь использовал в пароле только не повторяющиеся символы.
