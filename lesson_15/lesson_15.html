Block #15. Set в JavaScript

Task 1. Создайте новый Set. Добавmте в него через add элементы. Выведите в консоль.

Task 2. Создайте input куда пользователь может вводить элементы и кнопку. По нажатию на кнопку значение из input добавляется в set. Set выводится в консоль.

Task 3. Добавьте к предыдущему заданию кнопку удалить. При нажатии кнопки читайте содержимое input и удаляйте его из Set. Если input пустой - выводите ошибку (alert).
